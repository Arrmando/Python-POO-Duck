# Template de Conferência IEEE

Este diretório contém o template oficial da IEEE para artigos de conferência, adaptado para uso neste projeto.
Saiba mais: <https://www.ieee.org/conferences/publishing/templates>

O template original contém um arquivo `.cls` que foi omitido aqui,
já que a imagem docker utilizada para compilação já contém o pacote necessário.
A imagem que está sendo usada é a `ghcr.io/xu-cheng/texlive-debian`.
Saiba mais: <https://github.com/xu-cheng/latex-docker>
