# Projeto 2 - Campo Minado

Campo Minado é um jogo antigo, um classíco dos jogos digitais, e desde sua criação em 1989 ele foi recriado das maneiras mais criativas possíveis.
Existem versões com o campo em formatos diferentes (2D, 3D, 6D, Hexagonal etc.), e com objetivos únicos como a versão inspirada em RPG medieval.
Tendo isso em vista, o objetivo do nosso grupo é recriar o código do campo minado clássico, mas de maneira a tornar possível a sua utilização para criar versões mais criativas do jogo.

## Rodando o jogo

Pré-requisitos:

- Ter o [just](https://just.systems/man/en/installation.html) instalado
- Ter o [uv](https://docs.astral.sh/uv/getting-started/installation/) instatalado

Rode o jogo com:

```bash
just run
```

## Implementação

- Linguagem: Python
- Framework: Pygame

Em decorrencia da utilização do Pygame para aplicar o front-end, as expansões possíveis do código fonte terão foco no que pode ser abstraido ao 2D.
